## Synopsis

bioimagetools is an R package for working with images in 3d and 4d. 

## Installation

    install.packages(c("devtools","spatstat","tiff"))
    source("https://bioconductor.org/biocLite.R")
    biocLite("EBImage")
    devtools::install_github("volkerschmid/bioimagetools")
